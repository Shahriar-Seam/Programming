### Prerequisites

### Running Solutions

#### For C/C++ files:
```bash
# Compile and run a C++ solution
g++ -o solution solution.cpp
./solution < input.txt

# For competitive programming (fast I/O)
g++ -O2 -std=c++17 solution.cpp -o solution
```

#### For Python files:
```bash
python solution.py < input.txt
```
